What's being built next

Phase 1 — aktiv

Admin Panel — Phase 1 Abschluss

Changelog-Admin + Blog/Thoughts-Editor im Admin Panel. Letzter Schritt von Phase 1 bevor das Portfolio öffentlich geht.

Phase 2

Öffentliche Changelog-Seite

Eigene /changelog-Seite — alle Releases nach Version, Datum und Kategorie gelistet. Footer-Link, vollständig mehrsprachig.

Phase 2

Copy Email Button

Ein-Klick-Kopieren der E-Mail in der Kontaktsektion — direkt in die Zwischenablage, kein Mailto-Link.

Phase 2

GitHub Activity Widget

Live GitHub-Aktivitäten auf der Startseite — Contribution-Graph oder letzte Commits. Belegt aktives Development ohne ein Wort.

Phase 2

Demo-Video Integration

Kurze Demo-Videos direkt in Projektseiten — StudyNexus zuerst. Lazy-Loading, kein Autoplay, kein Performance-Impact.

Phase 2

Project Status Indicators

Live-Status-Badges auf Projektkarten — "In Entwicklung", "Beta", "Live". Gesteuert aus dem Admin Panel.

Geplant

Dark Mode

Light/Dark Mode Toggle unter Beibehaltung der Vanilla-Performance. Status-Speicherung via LocalStorage.

Geplant

Blog / Thoughts

Eigene /thoughts Sektion — kurze Posts, LinkedIn-Stil aber individuell. Verwaltet im Admin Panel.

Phase 3

Animierte Seitenübergänge

Smooth Seitenübergänge via View Transitions API — kein Framework, kein Layout-Shift, kein Performance-Impact.

Phase 3

ES Modules Migration

Umstieg auf native ES Modules im Browser — saubere Scope-Isolation, kein globaler Window-Clutter, ohne Bundler.

Geplant

Arabisch (AR) — Dritte Sprache

Vollständige AR-Übersetzung + RTL-Support. Language Switcher: DE → EN → AR.
